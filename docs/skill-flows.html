Skill Overview

6 Skills
/quote

Quote

Read-only. Fetches swap route and displays expected output, rate, and gas cost. No transaction built.

/swap-build

Swap Build

Fetches route, asks for confirmation, then builds encoded calldata. Does NOT submit on-chain.

/swap-execute

Swap Execute

Takes calldata from /swap-build and broadcasts it on-chain via cast send. Requires confirmation.

/swap-execute-fast

Fast Execute

Build + broadcast in one step. Shell script handles everything. Only one verbal confirmation before launch.

/limit-order-fast

Fast Limit Order

Signs EIP-712 and submits a gasless limit order immediately. No review step. USD threshold guard ($1,000). DSLOProtocol contract cross-check.

/zap-fast

Fast Zap

Builds and broadcasts a concentrated-liquidity zap-in immediately. Pre-flight balance + allowance checks. ZapRouter address verification.

Full Swap Lifecycle — How Skills Connect

Architecture
Two paths to execute a swap: the safe 2-step flow (build → execute) and the fast 1-step flow. Token resolution is shared across all skills.
