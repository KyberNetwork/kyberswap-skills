# Supported DEX Identifiers

71 DEXes supported by KyberSwap Zap as a Service (ZaaS). Use the DEX ID in the `dex` parameter for ZaaS API calls.

When a user provides shorthand (e.g., `uniswapv3`), normalize to API format (e.g., `DEX_UNISWAPV3`) before calling the API.
